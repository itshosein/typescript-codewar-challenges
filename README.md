### Installation

<div id="installation"></div>

1. Clone the repo

   ```sh
   git clone git@github.com:itshosein/typescript-codewar-challenges.git
   ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. For development

   ```sh
   npm run dev
   ```

   For production

   ```sh
   npm run build

   npm start
   ```
